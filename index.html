About GTSOFIA

About Us

GTSOFIA is a comprehensive public transportation information system for Sofia, Bulgaria. Our mission is to provide real-time information about buses, trams, trolleybuses, and metro lines to help residents and visitors navigate the city efficiently.

Features

  • View all public transportation lines in Sofia
  • See detailed stop information for each line
  • Switch between different directions
  • Search for specific lines
  • Filter by transport type (bus, tram, trolley, metro)
  • Responsive design for mobile and desktop

Technology

GTSOFIA is built with modern web technologies including HTML5, CSS3, and JavaScript. The application provides a responsive user interface that works seamlessly on desktop, tablet, and mobile devices.

Data

Our transportation data is maintained and updated regularly to ensure accuracy and relevance. All line information, stops, and directions are carefully catalogued in our comprehensive database.
